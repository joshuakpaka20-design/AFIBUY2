# Afibuy Trade Intelligence Platform

## Features
- User Registration & Login
- Password Hashing
- SQLite Database
- AI Forecasting
- Risk Analysis
- Opportunity Scoring
- Trade Alerts
- PDF Reporting
- Admin Analytics
- Advanced Visualizations
- Trade News Feed

## Run Locally

```bash
pip install -r requirements.txt
streamlit run afibuy_app.py
```

## Deploy to Streamlit Cloud
1. Upload files to GitHub
2. Connect repository to Streamlit Community Cloud
3. Select `afibuy_app.py`
4. Deploy
